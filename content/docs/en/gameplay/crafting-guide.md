# Crafting Guide - All Recipes

Welcome to the complete crafting guide for Hytale. This guide covers all known recipes, crafting stations, and tips to help you master the art of item creation in Orbis.

:::note Early Access
Hytale is currently in Early Access. Recipes and crafting mechanics may change as the game develops. This guide is based on available information and classic survival game mechanics.
:::

## Introduction to the Crafting System

Crafting is essential for survival and progression in Hytale. By gathering resources and combining them at various crafting stations, you can create tools, weapons, armor, and much more.

### Accessing the Crafting Interface

- **Inventory Crafting**: Open your inventory to access basic 2x2 crafting grid
- **Crafting Station**: Interact with a crafting station for advanced recipes
- **Recipe Book**: Browse available recipes based on discovered materials

### Crafting Stations Overview

Different items require different crafting stations. Here's a quick overview:

| Station | Primary Use |
|---------|-------------|
| Hand Crafting | Basic items, first tools |
| Workbench | Advanced tools, weapons, blocks |
| Forge | Metal weapons, armor |
| Furnace | Smelting ores, cooking food |
| Alchemy Table | Potions and magical items |

## Crafting Tips

### Beginner's Crafting Order

Follow this progression for efficient early game:

1. **Punch trees** - Get wood
2. **Craft planks and sticks** - Basic materials
3. **Wooden pickaxe** - Mine stone
4. **Stone tools** - Better efficiency
5. **Furnace** - Smelt ores
6. **Workbench** - Advanced recipes
7. **Iron tools and weapons** - Combat ready
8. **Armor** - Survive exploration

### Efficiency Tips

:::tip Optimize Your Production
- **Batch craft** - Make multiple items at once to save time
- **Organize materials** - Keep chests sorted by material type
- **Central crafting hub** - Build all stations in one location
- **Automate when possible** - Use hoppers and redstone for automatic smelting
:::

### Resource Management

| Strategy | Benefit |
|----------|---------|
| Mine in branches | Maximum ore exposure |
| Replant trees | Sustainable wood supply |
| Cook in bulk | Efficient fuel usage |
| Save diamonds | Use for essential tools first |

### Common Mistakes to Avoid

- **Don't craft gold tools** - Low durability, save gold for other uses
- **Don't ignore food** - Hunger affects regeneration
- **Don't skip armor** - Protection is essential for exploration
- **Don't waste diamonds** - Plan before crafting
