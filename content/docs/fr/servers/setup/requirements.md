# Configuration requise du serveur

Exigences matérielles et logicielles.

## Logiciels

- Java 25+
- Linux, Windows ou macOS

## Matériel (minimum)

| Composant | Spécification |
|-----------|------|
| CPU | 4 cœurs |
| RAM | 6-8 Go |
| Stockage | 20 Go SSD |

## Matériel (recommandé)

| Composant | Spécification |
|-----------|------|
| CPU | 6+ cœurs |
| RAM | 12-16 Go |
| Stockage | 50 Go NVMe |
