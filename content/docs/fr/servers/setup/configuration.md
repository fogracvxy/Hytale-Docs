# Configuration du serveur

Configurez votre serveur.

## server.properties

```properties
server-name=Mon Serveur
port=5520
max-players=50
view-distance=10
```

## Paramètres clés

| Paramètre | Description |
|---------|-------------|
| `server-name` | Nom affiché |
| `port` | Port réseau |
| `max-players` | Limite de joueurs |
| `view-distance` | Distance de rendu |
