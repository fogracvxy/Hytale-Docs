# Guide d'auto-hébergement

Hébergez sur votre propre matériel.

## Prérequis

- IP statique ou DNS dynamique
- Capacité de redirection de ports
- Disponibilité 24/7 (optionnel)

## Étapes

1. Installer Java 25+
2. Télécharger les fichiers du serveur
3. Configurer le pare-feu
4. Rediriger le port 5520 (UDP/QUIC)
5. Démarrer le serveur
